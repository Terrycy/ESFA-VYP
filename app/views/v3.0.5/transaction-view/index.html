Filter and export transactions
Help using this page
This page displays a list of payment transactions for funding you have received from the ESFA. You can view transactions from up to 3 years ago. On this page you can:
  • Use the date filter to view transactions for a custom date range for either pre or post-16 funding.
  • Use the transactions filter to search for specific transaction descriptions within the date range.
  • Use the academies filter to search for specific payments from one or more academies.
  • View your total payments received for the filters you have selected.
  • View your budget group summary information for the filters you have selected.
  • Use the arrows next to each field heading to sort data into ascending/descending order. Sort by date descending to return to default view.
